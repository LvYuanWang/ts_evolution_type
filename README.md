# ts_evolution_type

#### 介绍
ts类型演算(ts的三个关键字)

#### 三个关键字

- typeof

TS中的typeof，书写的位置在类型约束的位置上。

表示：获取某个数据的类型

当typeof作用于类的时候，得到的类型，是该类的构造函数

- keyof

作用于类、接口、类型别名，用于获取其他类型中的所有成员名组成的联合类型

- in

该关键字往往和keyof联用，限制某个索引类型的取值范围。
